TTS JSON export for the 1490 DOOM Company Builder

Fetch Company as JSON

GET /s/:code?tts=1 CORS: *

Returns the full company roster as JSON. Use this in your TTS script to fetch a player's company directly from their share link.

The :code is the 7-character identifier from the end of any share URL — e.g. 6i6fmdj from https://1490doomcompanybuilder.com/s/6i6fmdj.

Example:

const res  = await fetch('https://1490doomcompanybuilder.com/s/6i6fmdj?tts=1')
const data = await res.json()
// data.warriors → array of warrior objects

Returns 404 if the code does not exist.

Note: Short link codes are permanent. The same company state always produces the same code, so you can store a code as a stable reference to a specific roster.

TTS JSON Format

The response is a single JSON object with company metadata and a warriors array. The shape differs slightly between standard and campaign mode — campaign adds per-warrior progression fields and replaces ipLimit with campaignGame.

Standard mode example

{
  "mark":        "Tower Born",
  "companyName": "Iron Ravens Pack",
  "companyMode": "standard",
  "ipLimit":     3,          // total IP available to the company
  "warriors": [
    {
      "type":       "Brute",
      "name":       null,         // null = no custom name
      "isCaptain":  true,
      "weapon1":    "Heavy Weapon",
      "weapon2":    null,
      "consumable": "Concentrated Creeping Death Serum",
      "climbing":   null,
      "ipUpgrades": ["consumable"]
    },
    {
      "type":       "Assassin",
      "name":       "Rook",
      "isCaptain":  false,
      "weapon1":    "Bow",
      "weapon2":    null,
      "consumable": "Fog of War Flask",
      "climbing":   "Grappling Hook",
      "ipUpgrades": ["climbing", "consumable", "stat_VIT"]
    }
  ]
}

Campaign mode example

{
  "mark":         "Graveborn",
  "companyName":  "The Hollow March",
  "companyMode":  "campaign",
  "campaignGame": 4,      // which game in the campaign
  "warriors": [
    {
      "type":         "Knight",
      "name":         null,
      "isCaptain":    true,
      "weapon1":      "Heavy Weapon",
      "weapon2":      "Shield",
      "consumable":   null,
      "climbing":     null,
      "ipUpgrades":   ["weapon2", "stat_VIT", "stat_DEF"],
      "earnedIP":     3         // campaign only
    }
  ]
}
