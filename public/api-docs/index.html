Endpoints

Resolve a short link (browser)

GET /s/:code

Loads the company in the builder. Intended for sharing with players.

Example: https://1490doomcompanybuilder.com/s/6i6fmdj

Fetch company as JSON (TTS integration)

GET /s/:code?tts=1 CORS: *

Returns the company as JSON. Use this endpoint in your TTS script to fetch roster data directly.

Example fetch:

const res  = await fetch('https://1490doomcompanybuilder.com/s/6i6fmdj?tts=1')
const data = await res.json()
// data.warriors → array of warrior objects

Returns 404 if the code does not exist.

Note: Short link codes are permanent. The same company state always produces the same code, so you can store a code as a stable reference to a specific roster.

Create a short link

POST /api/s

Generates a short link from an encoded company hash. This is called automatically by the Share modal — you don't need this unless you're building a custom integration.

// Request
POST /api/s
Content-Type: application/json

{ "encoded": "NnxJcm9u..." }

// Response
{ "code": "abc1234", "url": "https://1490doomcompanybuilder.com/s/abc1234" }

TTS JSON Format

The top-level object contains company metadata and a warriors array. The shape differs slightly between standard and campaign mode — campaign adds per-warrior progression fields and replaces ipLimit with campaignGame.

Standard mode example

{
  "mark":        "Tower Born",
  "companyName": "Iron Ravens Pack",
  "companyMode": "standard",
  "ipLimit":     3,          // total IP available to the company
  "warriors": [
    {
      "type":       "Brute",
      "name":       null,         // null = no custom name
      "isCaptain":  true,
      "weapon1":    "Heavy Weapon",
      "weapon2":    null,
      "consumable": "Concentrated Creeping Death Serum",
      "climbing":   null,
      "ipUpgrades": ["consumable"]
    },
    {
      "type":       "Assassin",
      "name":       "Rook",
      "isCaptain":  false,
      "weapon1":    "Bow",
      "weapon2":    null,
      "consumable": "Fog of War Flask",
      "climbing":   "Grappling Hook",
      "ipUpgrades": ["climbing", "consumable"]
    }
  ]
}

Campaign mode example

{
  "mark":         "Graveborn",
  "companyName":  "The Hollow March",
  "companyMode":  "campaign",
  "campaignGame": 4,      // which game in the campaign
  "warriors": [
    {
      "type":         "Knight",
      "name":         null,
      "isCaptain":    true,
      "weapon1":      "Heavy Weapon",
      "weapon2":      "Shield",
      "consumable":   null,
      "climbing":     null,
      "ipUpgrades":   ["weapon2"],
      "earnedIP":     3,       // campaign only
      "statImproves": ["VIT", "DEF"] // campaign only
    }
  ]
}
